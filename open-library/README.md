# Open Learning Library

This folder contains the public-good assets that accompany the app:

- `rubrics/`: simulation-only rubric files that educators and builders can adapt
- `benchmark/`: starter benchmark manifests and contribution notes for labeled simulation imagery

The goal is to make the project useful beyond one UI by sharing the evaluation structure, not just the product shell.

In the app, these materials are surfaced through the `/library` route so learners, faculty, and collaborators can inspect the public assets without digging through the repo first.

## Principles

- Simulation-only scenes only
- Faculty-reviewable labels
- Plain JSON and CSV formats
- Safer, broader skills prioritized before flashier procedures

## Included Assets

- `rubrics/simple-interrupted-suture.json`
- `rubrics/rubric-template.json`
- `benchmark/simulation_benchmark_manifest.csv`
- `benchmark/README.md`

## Current Emphasis

- one live rubric for the simple interrupted suture trainer
- starter benchmark structure rather than a fully published image dataset
- simulation-only contribution norms that can scale to broader, safer skills over time
